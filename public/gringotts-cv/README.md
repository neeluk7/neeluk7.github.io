# Vault Ledger Resume

This is a self-contained Gringotts-inspired resume theme in `index.html`.

## Edit

Open `index.html` in any editor and replace the bracketed placeholders:

- `[Your Name]`
- `[Project Name]`
- `[Company or Lab Name]`
- `[University Name]`
- skills, dates, links, outcomes, and descriptions

## Export to PDF

Open `index.html` in a browser, then use Print:

- Destination: Save as PDF
- Paper size: Letter
- Margins: None or Default
- Background graphics: Enabled

The theme uses original inline clipart and CSS, so there are no external image files to manage.
